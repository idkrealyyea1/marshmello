 HOW TO DEPLOY TO GITHUB PAGES
──────────────────────────────────────────────────────────────

  1. Create a new GitHub repository (e.g. "grand-elysee")
  2. Upload index.html, style.css, script.js
     directly to the root of the repository
  3. Go to: Settings → Pages → Source → "Deploy from a branch"
  4. Select branch: main / root folder
  5. Click Save — live in ~60 seconds at:
     https://yourusername.github.io/grand-elysee/

  ✓ No build step required
  ✓ Works immediately on extract
  ✓ All images load from Unsplash CDN (no hosting needed)

 CUSTOMIZATION
──────────────────────────────────────────────────────────────

  BRAND NAME
  ──────────
  Search "Grand Élysée" in index.html and replace.

  COLORS (style.css :root block)
  ──────
  --gold:        #c9a963   (primary gold)
  --gold-light:  #e8d5a3   (highlight gold)
  --bg-deep:     #06060f   (darkest bg)
  --bg-dark:     #0a0a18   (dark bg)

  REPLACING IMAGES WITH YOUR OWN
  ────────────────────────────────
  Find any <img src="https://images.unsplash.com/...">
  and replace the src with your local path or URL:
    src="./images/my-photo.jpg"

  BOOKED CALENDAR DATES
  ──────────────────────
  In script.js, find:
    const bookedDays = [3, 7, 12, 14, 19, 21, 26, 28];
  Change numbers to match your actual booked dates.

  CONTACT DETAILS
  ───────────────
  Update address, phone, email in the #contact section.

 PERFORMANCE NOTES
──────────────────────────────────────────────────────────────

  • Zero JavaScript dependencies — no jQuery, no frameworks
  • CSS animations use transform/opacity only (GPU-accelerated)
  • IntersectionObserver for all scroll reveals
  • Images use loading="lazy" (except hero = eager)
  • Unsplash images use CDN parameters (w=, q=, auto=format)
  • Passive scroll listeners throughout
  • No layout-thrashing DOM reads in animation loops
  • will-change: transform on cursor and parallax elements
